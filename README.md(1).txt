👟 Adidas Sales Analysis (Power BI)

An interactive Power BI dashboard analyzing Adidas' U.S. sales by product type, region, time, and retailer.

📊 Key Metrics
- **Total Sales:** $900M | **Profit:** $332M | **Units Sold:** 2M | **Margin:** 42%
- Top-performing product: *Men's Street Footwear* ($0.21B)
- Top-performing region: *West* ($270M)
- Peak month: *August* ($95M)
- Top retailer: *West Gear* ($0.24B)

🛠️ Tools Used

- Excel/CSV data sources
